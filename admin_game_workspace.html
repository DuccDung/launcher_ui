Kho account

Bảng accounts là kho toàn cục; version của game đang chọn có thể trỏ vào bất kỳ account nào trong kho này.
accounts
Sau khi tạo account mới, bạn có thể quay lại form version để gắn ngay vào game hiện tại. File tải xuống sẽ bám theo account_id.

Builder bài article dễ dùng

Người dùng chỉ cần chọn box nội dung, chọn định dạng rồi nhập text/link/ảnh. Hệ thống sẽ tự convert thành content_json chuẩn ở phía dưới.
articles
Quy trình dễ hiểu cho user: 1. tạo box, 2. chọn loại nội dung, 3. nhập nội dung, 4. xem preview và JSON hệ thống tự sinh.
